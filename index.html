A propos de moi

Présentation

Je m'appelle Nicolas David, développeur et technicien dans l'évenementiel.
Je travaille actuellement chez Hardis Group, en tant qu'Ingénieur en développement informatique.
Je suis aussi technicien du collectif L'octopus, ou je tiens le rôle de technicien lumière, réseau, et développeur.
Depuis petit je suis intéressé par l'informatique, et je suis capable de développer des applications en full stack dans différentes technologies, principalement Java / Angular / C# / PHP / Unity / Python

Ce que je fais

Je développe les applications qui m'intéressent et m'épanouissent dans différentes technologies.
J'aime les défis, et essaye toujours de me perfectionner dans ce que je fait de mieux, mais souvent jamais assez à mon gout.
J'aime développer des applications dans différents domaines, dans un bureau comme dans une salle de spectacle, pour moi chaque application est unique et mérite qu'on y passe le temps nécessaire pour pouvoir en tirer le meilleur.

Un autre type de métier

A côté de mon travail de développeur, j'ai aussi développé une passion pour l'éclairage dans l'évenementiel.
Je suis donc capable d'effectuer un travail de mise en lumière d'évènement, de la conception 3D au pupitrage des différentes machines composant la scène.
Mes compétences me permettent donc de mettre en lumière un évènement, connaissant tout particulièrement la console lumière grandMA2, leader du marché actuellement, et permettant l'asservissement de toute les machines existantes sur le marché.

Les technologies que j'utilise le plus

Java - Spring boot

Java est pour moi le meilleur language pour un back office solide. Associé au framework spring boot, il permet de tirer le meilleur du language avec simplicité et efficacité

Angular

Angular est l'un des framework que je préfère pour construire une page web. Grâce au language de programmation Typescript, il permet simplement de construire une application complexe facilement maintenable, et offre des outils améliorant l'expérience utilisateur à tout les niveaux.

Python

Python, permet grâce à sa fiabilité et sa legèreté de s'installer facilement sur n'importe quel système. Je l'utilise principalement avec des raspberry, il permet de construire des programmes simples mais très utiles pouvant être embarqués de partout.
